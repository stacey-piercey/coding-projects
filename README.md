# coding-projects

A sandbox repository for learning by building.

This is where we figure things out. Each project lives in its own folder and grows into whatever it needs to be — an app, an agent, a website, a game. No fixed roadmap, just building, breaking, and getting better.

## What's in here

Each subfolder is its own self-contained project with its own purpose, stack, and README. Think of this repo as the container and each project as the thing inside it.

## Who this is for

Two people learning together. If you're new to the team, start with the [onboarding guide](./Claude-Code-Windows-Setup-Guide.docx) to get your environment set up.

## What we might build

* Web apps
* AI agents
* Websites
* Video games
* Whatever else seems interesting

## How we work

We use Claude Code as our AI coding agent and Git to track everything. Each project gets its own folder. When something grows big enough to need its own repository, we'll spin it off.
